累積次數分配表(cumulative frequency distribution table)，就是將[[次數分配表]]的次數由低到高逐各累加上去。
- - -
# 範例：
- 期中考班上分數的次數分配表

| 分數   | 次數  |               | 分數   | 累積次數 |
| ---- | --- | ------------- | ---- | ---- |
| 100  | 3   |               | 100  |      |
| 90   | 2   |               | 90   |      |
| 80   | 6   |               | 80   |      |
| 70   | 17  | $\rightarrow$ | 70   |      |
| 60   | 16  |               | 60   |      |
| 未滿60 | 6   |               | 未滿60 |      |
| 總計   | 50  |               | 總計   |      |

- 參加者學歷

| 學歷    | 次數  |               | 學歷    | 累積次數 |
| ----- | --- | ------------- | ----- | ---- |
| 博士    | 1   |               | 博士    |      |
| 碩士    | 0   |               | 碩士    |      |
| 大學(專) | 5   |               | 大學(專) |      |
| 高中(職) | 3   | $\rightarrow$ | 高中(職) |      |
| 國中    | 1   |               | 國中    |      |
| 國小    | 0   |               | 國小    |      |
| 國小以下  | 0   |               | 國小以下  |      |
| 總計    | 10  |               | 總計    |      |
