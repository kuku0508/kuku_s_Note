次數分配表(frequency distribution table)，通常是依照數值大小或類別種類而分成好幾組，並列出各組所含的次數。
- - -
# 範例：
- 期中考班上分數的次數分配表

| 分數   | 次數  |
| ---- | --- |
| 100  | 3   |
| 90   | 2   |
| 80   | 6   |
| 70   | 4   |
| 60   | 16  |
| 未滿60 | 6   |
- 參加者學歷

| 學歷    | 次數  |
| ----- | --- |
| 博士    | 1   |
| 碩士    | 0   |
| 大學(專) | 5   |
| 高中(職) | 2   |
| 國中    | 1   |
| 國小    | 0   |
| 國小以下  | 0   |
