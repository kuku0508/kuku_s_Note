次數分配表(frequency distribution table)，通常是依照數值大小或類別種類而分成好幾組，並列出各組所含的次數。
- - -
# 範例：
- 期中考班上分數的次數分配表

| 分數   | 次數  |
| ---- | --- |
| 100  | 3   |
| 90   | 2   |
| 80   | 6   |
| 70   | 4   |
| 60   | 16  |
| 未滿60 | 6   |
